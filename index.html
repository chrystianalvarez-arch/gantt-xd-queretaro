🔗 URL del Apps Script Web App Pega aquí la URL de tu Google Apps Script desplegado como Web App.
📋 Instrucciones de deploy
  1. Crea un Google Sheet nuevo en la carpeta de Drive del proyecto.
  2. Abre Extensions → Apps Script.
  3. Borra el contenido del editor y pega el contenido del archivo Code.gs.
  4. Guarda (Ctrl+S) y ejecuta onOpen una vez para verificar permisos.
  5. Haz clic en Deploy → New deployment.
  6. Tipo: Web App. Execute as: Me. Who has access: Anyone.
  7. Copia la URL generada y pégala arriba en "URL del Apps Script".
  8. Desde el menú 🚀 Gantt XD → Inicializar datos para cargar los datos por primera vez.
  9. Publica este index.html en GitHub Pages (repo nuevo, rama main, raíz /).
  10. Comparte la URL de GitHub Pages con el equipo.
⚠️ Modo offline

Si no hay URL configurada o la API no responde, el sistema trabaja en modo local (localStorage) y muestra un aviso. Los cambios locales no se sincronizan al reconectar automáticamente — usa el botón Recargar datos.
